Bienvenido a SuperMarket

RESUMEN

El objetivo de esta web es brindar la oportunidad a empleadores/empleados/gerentes/etc.
administrar, visualizar y almacenar datos sobre los productos de un minimercado enfocado a la venta de alimentos (Carne y Verduras).
Ademas de esto permite llevar un registro sobre el aumento mensual de cada producto y cantidad de los mismos.
